CONTROLES


Para moverte horizontalmente, utiliza las flechas de dirección derecha e izquierda.

Para saltar cuando quedas atrapado en la pared, utiliza la barra espaciadora.

En el aire, utilizando las flechas de dirección verticales, Dimitri puede adoptar una posición más aerodinámica para así aumentar su velocidad (flecha de dirección vertical inferior).

También puede reducir su velocidad de caída mediante el uso de una tela (flecha de dirección vertical superior).
